# ARIL 0.4.2 — In-chat diagrams and self-update

Point release that renders Mermaid / SVG / ASCII diagrams in assistant replies,
and adds `/update` so you can install the latest notarized DMG from GitHub
without leaving the app.

Builds are signed with Developer ID and **notarized by Apple** — no Gatekeeper
workaround needed.

## Install

1. Download **ARIL-0.4.2.dmg**
2. Open the disk image and drag **ARIL** into **Applications**
3. Launch ARIL (Solo mode starts the bundled gateway)
4. Open **Preferences → General** and paste your OpenRouter API key (if needed)

If you already have ARIL 0.4.1+, you can also run **`/update`** in chat to
download and install this release into `/Applications`.

## What's new in 0.4.2

### Diagrams in chat
- Assistant replies that include fenced **`mermaid`**, **`svg`**, or **`ascii`**
  blocks (plus inline `<svg>` and SVG image links) render in the message bubble.
- Mermaid uses a WebKit renderer (network needed once for the Mermaid library).
- SVG is sanitized before display; ASCII art gets a monospaced panel with copy.

### Self-update
- Slash command **`/update`** (alias `/upgrade`) checks GitHub Releases for a
  newer `ARIL-*.dmg`.
- If a newer version exists, ARIL prompts to upgrade, then downloads the DMG,
  replaces **`/Applications/ARIL.app`**, and relaunches.
- `/status` points you at `/update` when you’re behind the latest release.

## Requirements

- **Apple Silicon Mac** (M-series) with **macOS 14** (Sonoma) or newer
- An [OpenRouter](https://openrouter.ai/keys) API key with a small credit balance

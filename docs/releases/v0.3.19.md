## Requirements

- **Apple Silicon Mac** (M-series) with **macOS 14** (Sonoma) or newer
- An [OpenRouter](https://openrouter.ai/keys) API key (`sk-or-v1-…`) with a small credit balance

## Install

1. Download **ARIL-0.3.19.dmg**
2. Open the disk image and drag **ARIL** into **Applications**
3. Quit any running ARIL first so the Solo gateway can restart
4. Launch ARIL → **Preferences → MCP**

## What's new in 0.3.19

- MCP **in chat** — enabled remote servers are tools in Auto/Manual (status line in the reply)
- MCP **Check** recycles an outdated local gateway after upgrade (fixes DeepWiki 404)
- MCP **Edit** shows preset name, URL, and auth (locked fields no longer look empty)
- DeepWiki (no key) should report Connected with 3 tools when Check succeeds

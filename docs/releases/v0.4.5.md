# ARIL 0.4.5 — Themes and a living title mark

Point release that expands Appearance presets (including System), and gives the
title-bar ARIL wordmark a quiet flourish every minute.

Builds are signed with Developer ID and **notarized by Apple** — no Gatekeeper
workaround needed.

## Install

1. Download **ARIL-0.4.5.dmg**
2. Open the disk image and drag **ARIL** into **Applications**
3. Launch ARIL (Solo mode starts the bundled gateway)
4. Open **Preferences → General** and paste your OpenRouter API key (if needed)

If you already have ARIL 0.4.2+, you can also run **`/update`** in chat.

## What's new in 0.4.5

### Appearance
- **System** theme mirrors macOS light, dark, and Auto (palette switches with
  the OS).
- New fixed schemes: **Ocean**, **Graphite**, **Sand**, **Dusk**, **Midnight**
  (alongside Noir, Slate, Light, and Forest).

### Title bar
- The **ARIL** label is a custom wordmark that every ~60 seconds crossfades,
  blooms, sparkles, or shifts typeface/colour — Reduce Motion swaps styles
  without animation.

### Toolbar
- Scroll-to-bottom **↓** sits centered in the prompt bar overlay (Mode / Web
  stay left).

## Requirements

- **Apple Silicon Mac** (M-series) with **macOS 14** (Sonoma) or newer
- An [OpenRouter](https://openrouter.ai/keys) API key with a small credit balance

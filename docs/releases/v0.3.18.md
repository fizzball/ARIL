## Requirements

- **Apple Silicon Mac** (M-series) with **macOS 14** (Sonoma) or newer
- An [OpenRouter](https://openrouter.ai/keys) API key (`sk-or-v1-…`) with a small credit balance

## Install

1. Download **ARIL-0.3.18.dmg**
2. Open the disk image and drag **ARIL** into **Applications**
3. Launch ARIL and open **Preferences → MCP** to configure servers

## What's new in 0.3.18

- **Preferences → MCP** — enable remote presets (Agenty, AI Diagram Maker, Cloudflare Browser, DeepWiki, GitHub, Firecrawl)
- API keys stored in Keychain; **Check connection** via Solo gateway
- Add / edit custom remote HTTP MCP servers
- Playwright listed as deferred (local Node / stdio later)
- Chat does not invoke MCP tools yet (config-only phase)

# ARIL 0.4.4 — Accurate session restore and scroll-to-bottom

Point release that stops duplicated image turns when restoring chat history,
and adds a toolbar control to jump to the latest message.

Builds are signed with Developer ID and **notarized by Apple** — no Gatekeeper
workaround needed.

## Install

1. Download **ARIL-0.4.4.dmg**
2. Open the disk image and drag **ARIL** into **Applications**
3. Launch ARIL (Solo mode starts the bundled gateway)
4. Open **Preferences → General** and paste your OpenRouter API key (if needed)

If you already have ARIL 0.4.2+, you can also run **`/update`** in chat.

## What's new in 0.4.4

### Session history
- Client and gateway **dedupe** restored transcripts so user/assistant pairs
  (especially turns with images) are not duplicated on launch.
- Prefer richer copies (`file://` images) over empty or placeholder replies.
- Existing caches are healed automatically on the next start.

### Scroll to bottom
- Circular **↓** button next to **Web** (same style as the send ↑ button)
  scrolls the message list to the latest turn.

## Requirements

- **Apple Silicon Mac** (M-series) with **macOS 14** (Sonoma) or newer
- An [OpenRouter](https://openrouter.ai/keys) API key with a small credit balance

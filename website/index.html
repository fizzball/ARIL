Features

Built for power users who care about cost and quality

ARIL runs a local gateway on your Mac, connects to OpenRouter, and helps you choose the right model for every task — without leaving the desktop.

Auto · Manual · Judge

Let ARIL route by category, lock a model, or compare three providers side-by-side and pick a winner.

Prompt intelligence

Grade, token estimate, latency probe, and rewrite suggestions while you type — before you spend.

Cost transparency

Per-message cost footers, session totals, budget guardrails, and OpenRouter credit status in the footer.

Learning loop

Prefer a Judge winner and Auto promotes that model next time for the same kind of prompt.

Local guardrails

Redact sensitive patterns and block prompt-injection attempts before they leave your Mac.

MCP tools

Built-in nmap and Semgrep scanners over MCP, plus slash commands for status, scans, and updates.

In the app

See it working

Intelligence before you send

Prompt grade, category, confidence, cost estimate, and an improved prompt alternative — all in one panel.

Clear, streaming replies

Markdown and math render legibly. Every reply shows the model, tokens, and exact OpenRouter cost.

Judge three models at once

Run the same prompt across GPT, Llama, and Gemini. ES scores help you select your preferred answer, and ARIL will automatically use that model next time a similar prompt is run.

Images, logs, and tools

Generate images in chat, inspect OpenRouter transactions live, and run nmap or code scans via MCP.

Live log analysis

OpenRouter calls, token counts, and per-request cost — streamed in a toolbar panel so you always know what ran.

Managed MCP scanners

Local nmap and Semgrep servers launch automatically. Slash commands for status, scans, and updates.
